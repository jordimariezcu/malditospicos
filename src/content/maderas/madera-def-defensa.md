**Qué son las maderas DEF (defensa):**
Palas pensadas para jugadores defensivos. Son lentas, ligeras y ofrecen mucho control. Se usan para cortar, bloquear y devolver con precisión.

Ideal para:
- Estilo defensivo clásico o cortador.
- Jugadores que usan picos largos o antitop.
- Control máximo y pelotas molestas para el rival.

Ventajas:
- Muchísimo control.
- Muy buena absorción de velocidad.
- Permite colocar la pelota con precisión.

Desventajas:
- Muy poca velocidad.
- Difíciles de usar en ataque o contraataque.
- Requiere buena lectura del juego y paciencia.

Ejemplos:
Butterfly Innershield Layer ZLF
Stiga Defensive Classic
Donic Defplay Senso V3
TSP Balsa 3.5
